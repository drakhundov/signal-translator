# Signal translation algorithms used in computer communication.
| Input Data        | Output Signal           | Mode            |
| ----------------- | ----------------------- | --------------- |
| Digital → Digital | Digital line coding     | NRZ, Manchester |
| Digital → Analog  | Modulation              | ASK, FSK, PSK   |
| Analog → Digital  | Sampling + quantization | PCM, Delta      |
| Analog → Analog   | Analog modulation       | AM, FM          |
